# 🏛️ Freedes AI Render search

Інтелектуальна система для архітекторів, яка шукає рендери у вашій локальній базі за візуальною схожістю та текстовими описами.

## 🚀 Як користуватись:
1. Встановіть [Python 3.10+](https://www.python.org/downloads/).
2. Скачайте цей репозиторій.
3. Додайте свої зображення у папку `my_renders`.
4. Запустіть файл **`run.bat`**.

🛠️ Перший запуск та зручність

 Автоматичне налаштування: При першому відкритті файл run_arch_ai.bat сам встановить усі необхідні бібліотеки (requirements.txt) та завантажить ШІ-моделі (~2 ГБ). Це займе кілька хвилин, наступні запуски будуть миттєвими.
Ярлик на Робочий стіл: Для зручності ви можете натиснути правою кнопкою миші на файл run_arch_ai.bat -> Надіслати (Send to) -> Робочий стіл (Створити ярлик). Тепер ви зможете запускати пошук одним кліком.

## 💡 Поради для кращого пошуку:
* При першому запуску програма завантажить ШІ-моделі (~2 ГБ). Зачекайте завершення.
* Для текстових фільтрів краще використовувати англійську мову (напр. *modern facade, wood texture*).
* Натисніть кнопку **"Переіндексувати все"** у меню зліва після кожного додавання нових файлів.
