# 🏛️ Freedes AI Render search

Інтелектуальна система для архітекторів, яка шукає рендери у вашій локальній базі за візуальною схожістю та текстовими описами.

## 🚀 Як користуватись:
1. Встановіть актуальну версію [Python 3.10+](https://www.python.org/downloads/).
2. Скачайте цей репозиторій.
3. Додайте свої зображення у папку `my_renders`.
4. 🔑 Налаштування доступу (Hugging Face Token)

Для завантаження інтелектуальних моделей (DINOv2) необхідно використовувати власний безкоштовний ключ (токен) від Hugging Face.

   Перейдіть за посиланням https://huggingface.co/settings/tokens/ .
   Зареєструйтесь (якщо ще не маєте акаунта) та натисніть New token.
   Виберіть тип Read, скопіюйте створений токен (починається на hf_...).
   Відкрийте файл app_arch.py будь-яким текстовим редактором (Блокнот, VS Code).
   Знайдіть 15 рядок та вставте ваш токен:
   Python

    os.environ["HF_TOKEN"] = "ВАШ_ТОКЕН_ТУТ"
    
5. Запустіть файл **`run.bat`**.

🛠️ Перший запуск та зручність

 Автоматичне налаштування: При першому відкритті файл run_arch_ai.bat сам встановить усі необхідні бібліотеки (requirements.txt) та завантажить ШІ-моделі (~2 ГБ). Це займе кілька хвилин, наступні запуски будуть миттєвими.
Для зручності ви можете натиснути правою кнопкою миші на файл run_arch_ai.bat -> Надіслати (Send to) -> Робочий стіл (Створити ярлик). Тепер ви зможете запускати пошук одним кліком.

## 💡 Поради для кращого пошуку:
* При першому запуску програма завантажить ШІ-моделі (~2 ГБ). Зачекайте завершення.
* Для текстових фільтрів краще використовувати англійську мову (напр. *modern facade, wood texture*).
* Натисніть кнопку **"Переіндексувати все"** у меню зліва після кожного додавання нових файлів.
