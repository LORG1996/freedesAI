🏛️ Freedes AI Render Search v2.0

Інтелектуальна екосистема для архітекторів та візуалізаторів. Система дозволяє миттєво знаходити потрібні ракурси, матеріали або об'єкти у вашій локальній базі рендерів, використовуючи комбінацію нейромереж: CLIP (розуміння тексту), DINOv2 (візуальна схожість) та YOLOv8 (розпізнавання об'єктів).
✨ Ключові можливості:

    🔍 Гібридний пошук: Шукайте за картинкою, за текстом або комбінуйте обидва методи (наприклад: завантажте фото вітальні + текст "night lighting").

    🎯 Розумне кадрування (Cropper): Виділяйте конкретну частину рендера (наприклад, тільки крісло або текстуру стіни), і система знайде схожі деталі в інших проектах.

    🤖 Автоматичне розпізнавання об'єктів: Завдяки YOLOv8 система при індексації "бачить" окремі предмети на рендерах і дозволяє знаходити їх, навіть якщо вони займають лише частину кадру.

    ⚡ Інкрементальна індексація: Програма розуміє, які файли вже є в базі, і обробляє лише нові додані зображення. Це економить години часу на великих архівах.

    🔗 Підтримка ярликів (.lnk): Вам не потрібно копіювати гігабайти рендерів у папку програми. Просто закиньте ярлики на ваші робочі сервери чи диски у папку my_renders.

    ☁️ Синхронізація з Miro (Beta): Можливість автоматично витягувати рендери з ваших Miro-дощок у максимальній якості (HD).

🚀 Швидкий старт:

    Встановлення: Переконайтесь, що у вас є Python 3.10+.

    Підготовка бази: Скопіюйте рендери (або ярлики на папки з ними) у директорію my_renders.

    Ключ Hugging Face:

        Створіть безкоштовний токен на huggingface.co/settings/tokens.

        Вставте його у файл app_arch.py у рядок os.environ["HF_TOKEN"] = "ВАШ_ТОКЕН".

    Запуск: Просто запустіть run.bat. При першому запуску програма сама встановить бібліотеки та завантажить ШІ-моделі (~2.5 ГБ).

🛠️ Робота з Miro (Тестова функція):

Ви можете масово завантажувати оригінали зображень з Miro, ввівши Board ID та API Token.

    ⚠️ Важливо: Наразі функція працює у тестовому режимі. Зображення можуть завантажуватися з некоректними назвами (генеровані ID), оскільки API Miro не завжди віддає оригінальні назви файлів. Ми працюємо над покращенням цього модуля.

💡 Поради для архітектора:

    Мова запитів: Для текстового пошуку використовуйте англійську (наприклад: concrete wall, minimalist kitchen, panoramic window).

    Вага тексту: Використовуйте слайдер "Вага тексту", щоб збалансувати, що важливіше — візуальна схожість з прикладом чи ваше текстове уточнення.

    Оновлення бази: Після додавання нових проектів натискайте кнопку "Оновити базу (Тільки нові)". Програма швидко просканує зміни, не переробляючи старі фото.

    Результати: Система видає до 50 найкращих збігів за спаданням схожості.

Система розроблена для пришвидшення пошуку референсів та технічних рішень у великих архітектурних архівах.
