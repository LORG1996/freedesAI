# 🏛️ Freedes AI render search

Система розумного пошуку по архітектурним рендерам.

## 🚀 Як запустити:
1. Встановіть **Python 3.10** або новіше.
2. Скачайте цей проект (Зелена кнопка **Code -> Download ZIP**).
3. Розпакуйте і закиньте свої рендери у папку **my_renders**.
4. Запустіть файл **run_arch_ai.bat**.

## 🛠️ Що станеться при першому запуску:
* Скрипт сам встановить потрібні бібліотеки (PyTorch, Transformers, Streamlit).
* Автоматично завантажаться ШІ-моделі (~2 ГБ). Це потрібно зробити лише один раз.
## 💡 Поради:
* Використовуйте **англійську мову** у текстовому фільтрі для кращих результатів.
* Не забувайте натискати кнопку **"Оновити базу"** після додавання нових фото.
