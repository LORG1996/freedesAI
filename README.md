🏛️ ArchSearch AI Ultra

Інтелектуальна система пошуку по архітектурним рендерам. Цей інструмент дозволяє знаходити потрібні кадри у вашій базі за допомогою візуальних прикладів та текстових описів (на основі моделей CLIP та DINOv2).
🚀 Швидкий старт (Windows)

    Підготовка: Переконайтеся, що у вас встановлено Python 3.10+.

    Завантаження: Скачайте цей проект (Code -> Download ZIP) та розпакуйте його.

    Ваші рендери: Скопіюйте ваші зображення (JPG, PNG) у папку my_renders.

    Запуск: Просто запустіть файл run_arch_ai.bat.

🛠️ Перший запуск та зручність

    Автоматичне налаштування: При першому відкритті файл run_arch_ai.bat сам встановить усі необхідні бібліотеки (requirements.txt) та завантажить ШІ-моделі (~2 ГБ). Це займе кілька хвилин, наступні запуски будуть миттєвими.

    Ярлик на Робочий стіл: Для зручності ви можете натиснути правою кнопкою миші на файл run_arch_ai.bat -> Надіслати (Send to) -> Робочий стіл (Створити ярлик). Тепер ви зможете запускати пошук одним кліком.

🔍 Як користуватися

    У відкритому вікні браузера натисніть кнопку "🔄 ПЕРЕІНДЕКУВАТИ ВСЕ" (це потрібно робити щоразу, коли ви додаєте нові файли в my_renders).

    Завантажте картинку-приклад та виділіть потрібну область (вікно, матеріал, деталь фасаду).

    Використовуйте Текстовий фільтр (краще англійською, напр. brick, night, glass) та слайдер ваги, щоб уточнити пошук.
